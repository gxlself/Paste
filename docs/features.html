Feature tour

Features

History, search, pinboards, optional iCloud sync — Mac menu bar app, iOS app, custom keyboard. MIT, local-first.

Core ideas

What you get

KEEP · SEARCH · ORGANIZE · Everywhere — history, find, pinboards, optional sync across Mac & iOS.

KEEP

Unlimited history (within your rules)

Text, RTF, images, and file paths on macOS. SHA-256 dedup, timeline groups, retention presets and max item cap. Optional skip for excluded apps.

SEARCH

Find anything you copied

Live search, type tabs, regex quick filters, and custom categories.

ORGANIZE

Pinboards & Paste Stack

Multiple pinboard slots; queue clips and paste in order; plain-text options.

EVERYWHERE

Mac · iPhone · Keyboard

App Group + optional iCloud (same Apple ID) across Mac and iOS.

Trust

Sync · Privacy

iCloud (optional)

Same Apple ID on Mac and iOS. App Group links iOS app and keyboard on device.

Local-first

Sync off by default. Excluded apps, retention, clear history — your call.

Details

Behavior & UI

Assisted Paste, Paste Stack, excluded apps, preview, shortcuts, iOS keyboard.

Paste modes

Assisted Paste, Paste Stack, plain text vs rich. Paste multiple items in order from the stack.

Custom rules

Exclude apps; skip images; iOS rule toggles where applicable.

Preview and edit

Quick look, link previews, edit/rename text, multi-select, undo delete.

Panel

Menu bar, overlay position, light/dark/system, hotkeys.

iOS Keyboard

Paste Keyboard: search and insert clips while typing. Full Access for shared DB.

Shortcuts-ready

Global hotkeys, quick-paste by number. (Siri Shortcuts: use system automation as needed.)

一个项目，三种形态

同一个 Xcode 工程产出：Mac 菜单栏应用、iOS 伴侣应用与自定义键盘扩展。

Paste (macOS)

菜单栏应用：剪贴板历史、全局快捷键、悬浮面板、Paste Stack、Pinboards，可选 iCloud 同步。

macOS 13+ · 默认 ⌘⇧V

Paste-iOS

在 iPhone/iPad 上浏览同一套历史：Pinboards、筛选、分类、多选、扫描与快速新建文本。

iOS 16+ · App Group

Paste-Keyboard

自定义键盘：在邮件、备忘录等应用中直接选取已保存片段插入，无需切出。需开启「完全访问」。

亮点

留存 · 搜索 · 整理 · Mac / iPhone / 键盘。功能一览 →

留存

Mac 上完整历史：文本、RTF、图片、路径。去重、保留策略、时间线、排除应用。需要的内容随时能找到。

搜索

实时搜索、类型标签、正则预设、自定义分类。几秒内找到早上复制的那段内容。

整理

Pinboards、Paste Stack、置顶、纯文本粘贴选项。可选 iCloud 在 Mac 与 iOS 间同步。

  • macOS：文本、RTF、图片、文件路径 · SHA-256 去重 · 排除应用 · 保留与时间线 · 搜索、正则、自定义类型 · 辅助粘贴（需无障碍权限）
  • 同步：iOS 应用与键盘通过 App Group；macOS 与 iOS 通过可选 iCloud（同一 Apple ID）
  • 隐私：本地优先；iCloud 可选；所述配置下无统计与第三方 SDK

快速开始

下载（macOS）
最新构建：GitHub Releases — 下载 Paste-*-macos.zip，解压后将 Paste 拖入「应用程序」。首次打开若被拦截，右键 Paste → 打开（或在系统设置 → 隐私与安全性 中允许）。
下载（iOS）
Paste G 在 App Store — 适用于 iPhone 与 iPad。正式版付费；TestFlight 仍可参与测试。
克隆并打开
git clone https://github.com/gxlself/paste.git && cd paste && open Paste.xcodeproj
macOS（从源码运行）
选择 Paste scheme → My Mac → ⌘R。若使用辅助粘贴或全局快捷键，需在「系统设置 → 隐私与安全性 → 无障碍」中授权。
iOS 与键盘
签名与 App Group group.gxlself.paste-tool 见 SETUP.md。

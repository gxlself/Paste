FAQ

Mac app won’t open (unidentified developer)

Right-click Paste → Open, or allow in System Settings → Privacy & Security. Prefer a signed notarized build from a trusted distributor.

Keyboard shows no history on iOS

Enable Full Access for Paste Keyboard in Settings → General → Keyboard. Confirm App Group matches SETUP.md.

Assisted Paste / hotkeys don’t work

Grant Accessibility for Paste in System Settings. See README → Preferences → General.

Sync between Mac and iPhone

Enable iCloud sync in Paste preferences on both devices; same Apple ID. iOS app and keyboard always share via App Group locally.
